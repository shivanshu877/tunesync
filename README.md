# TuneSync

Sync YouTube Music playback across multiple Macs on the same Wi-Fi. No servers, no accounts, anyone can control.

## Build

```bash
swift build
```

## Run (dev)

```bash
swift run TuneSync
```

## Build .app bundle

```bash
make bundle
open TuneSync.app
```

## Test

```bash
swift test
```
